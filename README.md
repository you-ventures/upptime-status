# You! Ventures Status

⬆️ Uptime monitor and status page for You! Ventures services, powered by [Upptime](https://github.com/upptime/upptime).

## Monitored Services

| Service | URL |
|---------|-----|
| AgentAIShield | https://agentaishield.com |
| AAIS App | https://agentaishield.com/app.html |
| AAIS API Auth | https://agentaishield.com/api/auth/me |
| AAIS Ingest API | https://agentaishield.com/api/v1/ingest |
| Apex AI | https://apex-ai-production.up.railway.app |

## Status Page

📊 View live status at: https://you-ventures.github.io/upptime-status/

## How It Works

- GitHub Actions checks all endpoints every 5 minutes
- Downtime triggers automatic GitHub Issues
- Response times tracked with historical graphs
- Status page hosted via GitHub Pages (free, no server needed)
